# Conecta PR Assets

Screenshots and media for [`conecta-gov-frontend`](https://github.com/naldomadeira/conecta-gov-frontend) PRs.

This repo exists because the main repo is private — GitHub's `raw.githubusercontent.com` URLs require auth for private repos, breaking inline images for non-authenticated viewers and CI tooling. Public images go here so PR comments can reference them via stable raw URLs.
